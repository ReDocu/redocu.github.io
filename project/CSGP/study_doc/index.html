콘솔 게임으로 배우는 C · C++

변수 하나에서 시작해, 게임 회사 운영 게임과 실시간 게임 7종을 만들며 배우는 프로그래밍. 문법과 기술을 배우는 순서가 곧 무언가를 완성해 가는 순서입니다.

1부 · C 기본 문법 — 게임 회사 운영 게임

  • 001부 — C 기본 문법: 회사를 키우며 배우는 언어
  • 01A 모듈 · 1–4장 — 값·계산·분기: 회사에 숫자를 세우다
  • 02B 모듈 · 5–6장 — 반복문과 함수: 회사에 심장을 달다
  • 03C 모듈 · 7–8장 — 배열과 문자열: 회사에 사람과 이름을 채우다
  • 04D 모듈 · 9–10장 — 구조체와 포인터: 데이터에 모양을 주다
  • 05E 모듈 · 11–12장 — 동적 메모리와 랜덤: 회사가 자라고, 시장이 살아난다
  • 06F 모듈 · 13–15장 — 파일·모듈·캡스톤: 회사를 저장하고, 게임을 출시하다
